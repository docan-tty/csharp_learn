####  条件

##### 作用

让顺序执行的代码，产生分支， 产生逻辑变化

##### if语句

语法

> if(bool类型)
> {
>   满足条件要执行的代码，if代码块中}
> else{
>   不满足条件执行的代码
> }

**嵌套使用同理**

*if ... else if ... else 语句*

> if(bool类型)
> {
>   满足条件要执行的代码，if代码块中}
> else if（bool）{
>   满足
> }
> else{
>   不满足
> }
