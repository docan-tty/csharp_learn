#### 字符串拼接

##### 方式1

  +拼接

```
str += "" + 1 +2 +3 ; // 后面数字 toString

str += 1+ 2 + "" +( 3+ 4); //括号优先级，先计算优先级  拼接7
```
*注：只能单个+或 +=*


##### 方式2 

**语法：**  string.Format("代拼接内容"， 内容1，内容2，....)

想要拼接的内容用占位符替代{数字}，0~n一次往后

```
string.Format("我是{0},今年{1}岁","ty", 22);
```

##### 控制台打印拼接

*注意： 后面的内容比占位符多， 不会报错，多的内容丢失*
*后面的内容比占位符少，会报错*

```
Console.WriteLine("{0}{1}{2}",1 , true, false);
```

#### 条件运算符

**用于比较两个变量或常量**

">  == !=等"

*注意 存在两边的内容*

*必须使用比较的结果*

```
int a=1, b=2;
Console.Write(a > b);
```

**写法**

```
a > b;
a > 10;

result = a + 2 > a -3 + 2;
```
*先计算条件运算符左右，先计算，再比较*


##### 不能进行范围比较

判断是否在某两个值之间
