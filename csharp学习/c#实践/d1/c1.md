#### 控制台相关

##### 输入相关

```
char c = Console.ReadKey(true).KeyChar;
// 不会输入到控制台
```

##### 其他方法

```
Console.Clear(); // 清除控制台内容

// 控制台大小 窗口大小 < 缓冲区大小

Console.SetWindowSize(100, 50);

// 缓冲区

Console.SetBufferSize(1000, 500);

```

*设置光标位置， 左上角顶点为 00， 右侧为x轴正方向，下方为y轴正方向*

**横纵距离单位不容：y = 2x**

```
Console.SetCursorPosition(10, 5);
Console.WriteLine("1");
```
